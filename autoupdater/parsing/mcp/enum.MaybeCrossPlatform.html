pub enum MaybeCrossPlatform<T> {
    Cross(T),
    NotCross(HashMap<Platform, T>),
}
Expand description

This enum allows us to represent a T that may or may not depend on the platform that we wish to install for.

Variants

Cross(T)

This T is the same across all platforms.

NotCross(HashMap<Platform, T>)

This T depends on the Platform
